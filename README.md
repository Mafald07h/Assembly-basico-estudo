# Assembly-basico-estudo
Neste repositório, tento aprender o básico de assembly na arquitetura x86_64 da Intel e trazer um pouco dessa trajetória, isso será algo mais como hobby e sem o intúito de deixar isso como algum projeto ultra oficial.

## A coisa básica para se entender Assembly
Os programas em assembly são para serem executados em microprocessadores
A linguagem serve para se entender com clareza como o computador funciona. Básicamente, a linguagem assembly possui alguns objetivos simples, o primeiro é a capacidade de se realizar operações matemáticas e também operações lógicas, mover dados do processador, realizar operações de entrada e saida e realizar controle e desvio de condicionais.

## Registradores

- **Gerais:** Responsável pelo armazenamento de dados e a sua movimentação ocorre da memória para o processador.

- **Especiais:** São utilizados em execuções de instruções.

## Código ASCII

Código lançado em 1968 com o intuito de poder substituir o Baudot. O ASCII podia ter entre 0 e 127 valores, enquanto o Baudot possuia apenas 5 deles. Desses 128 simbolos, cada um deles se dividem em duas categorias distintas

- _Caracteres imprimiveis:_ (Letras,números,símbolos ...) São ao todo 96, que vai do 32 até o 127 símbolo

- _Caracteres não imprimiveis:_ (salto, linha, ...) São 31 símbolos, que vai do 0 até o 31.
